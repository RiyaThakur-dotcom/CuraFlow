## 🌟 Introduction

Welcome to **CuraFlow**! This isn't just another clinical dashboard. It's an application crafted with empathy, warm greetings, and emotional micro-copy to make clinic management feel less like a chore and more like an extension of the healing journey. 

Built as a lightning-fast, single-page React application, CuraFlow operates entirely in the browser using `localStorage`. No backend is needed—just open it up and let the care begin.

---

## ✨ Features

- 👋 **Dashboard**: Your clinic's pulse at a glance. Features dynamic time-based greetings, instant analytics stats, recent logs, and a heartfelt "Patient Spotlight".
- 👥 **Patients**: Searchable patient directories. Add new patients along with their personal stories and archive records safely when needed.
- 🩺 **Doctors**: Specialized healer cards displaying their unique mottos and live availability statuses.
- 📅 **Appointments**: Seamless scheduling with zero confusion. Book, complete, or cancel appointments using empathy-driven UI confirmations.
- 📝 **Medical Records**: An interactive timeline UI to document every chapter of a patient’s care journey, complete with diagnosis forms and doctor notes.
- 💝 **Donations**: A rich, engaging donations page to fund those in need, featuring preset amounts, live progress bars toward a ₹50,000 goal, and celebratory confetti upon donation.
- 📊 **Analytics Board**: Real-time visual data on patient demographics, top benefactors, and weekly appointment loads using custom-built mini bar charts.

---

## 🛠️ Tech Stack

CuraFlow relies on modern, lightweight frontend tools:
- **React.js** (useState, useEffect, useContext)
- **Vite** for blazing fast builds
- **Tailwind CSS** for stunning, responsive glassmorphic styling
- **Lucide-React** for crisp, elegant iconography
- **LocalStorage Data Persistence** for seamless offline state saving

## 🚀 Getting Started

To run CuraFlow on your local machine:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/RiyaThakur-dotcom/CuraFlow.git
   cd CuraFlow
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm run dev
   ```

4. **Experience the app**:
   Open **`http://localhost:5173`** in your browser and start managing care with heart.
